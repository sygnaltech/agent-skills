## Gitbook Repository Structure

Gitbook repos follow a specific structure. 


- SUMMARY.md is the left-side table of contents.  All pages must be listed here.
- Most other folders and .md pages are content and structure


### Assets

Images and other file assets go beneath `/.gitbook/assets` 


## Basic Operations 

### Adding Pages

1. Examine the structure of the repository in SUMMARY.md 
2. Determine where the Page should exist in the hierarchy 
3. Create the markdown page where it best fits 
4. Make certain to list the page in the SUMMARY.md in the appropriate location
